HLL LIVE PANEL

Overlay live pour Hell Let Loose créé par [CHM3] TONTONCAROTTE.

📌 Description

HLL LIVE PANEL est un overlay Electron permettant d’afficher en temps réel les statistiques de jeu sur Hell Let Loose.

Le logiciel fonctionne avec :

Steam
Epic Games
Gamepass

Le pseudo joueur est entièrement personnalisable directement depuis l’overlay.
